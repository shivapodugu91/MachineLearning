Steps to execute:

1)copy the src/ML4.java file into your local path
2) Uploaded the used 5 subfolders in each train and test data set.
3)Open Command Prompt/Putty and navigate to the src folder 
4)compile the program using the below command
 javac ML4.java
5)Our program is taking two arguments train_dataset path and test_dataset path. 
5)Run the program using the below command
java ML4 train_dataset_path test_dataset_path (include only 5 subfolders in each dataset)
for example: 
java ML4 C:\Users\shiva\Documents\Assignments\MI\New\tRAIN C:\Users\shiva\Documents\Assignments\MI\New\TEST
